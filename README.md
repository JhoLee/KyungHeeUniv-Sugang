# Kyung Hee University Sugang Master

## 🎓 How to use? 🎓
1. [경희대학교 수강신청 사이트](http://sugang.khu.ac.kr/)에 접속합니다.
2. 설치한 확장프로그램 아이콘을 **클릭**하여 실행합니다.
3. F5, ENTER, SPACEBAR 등의 키 금지 기능을 무효화했습니다.
4. 끝!
